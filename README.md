## how to use
<br>
### Just use it like jQuery :

``` javascript
#('#id').hide();
#('.class').show();
#('by tag').animate();
```
Note: you must first enable animation by<br>
``` Javascript
#().enableAnime();
```
Which enables it for all element on the page<br>
Then enable it for an element
``` JavaScript
#('.body').animate('jackInTheBox');
```
